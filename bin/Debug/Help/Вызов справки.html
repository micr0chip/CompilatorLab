File

  • Create - создаёт новый текстовый файл
  • Open - открывает текстовый файл
  • Save - сохраняет текстовый файл
  • Save as - сохранить текстовый файл в определенном формате
  • Exit - выход из приложения
    •

    Edit

  • Undo - отменить последнее изменение
  • Redo - отменить отменённое изменение
  • Cut - вырезать выделенный текст в буфер обмена
  • Copy - копировать выделенный текст в буфер обмена
  • Paste - вставить текст из буфера обмена
  • Delete - удалить выделенный текст
  • SelectAll - выделить весь текст
